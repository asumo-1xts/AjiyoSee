# What's this?

## **AjiyoSee（読み：あじよすぃー）**

つくば市の名店「あじよし」の営業状況をリアルタイムでお知らせする公認プロジェクトです。

平日に[公式X](https://twitter.com/ajiyoshiver2)の投稿を自動で読み取り、以下のいずれかを判定します。

| 営業状況 | 判定時刻 | 判定ワード |
| :------ | :------ | :----- |
| ⭕ **OPEN** | 17時 | ***メニュー*** \| ***オープン*** |
| ❌ **CLOSE** | 17時 | ***お休み*** |
| 🚫 **SOLD OUT** | 20時半 | ***完売*** |
| ❓ **UNKNOWN** | 17時 | 上記に該当なし |
| ⚠️ **X-API ERROR** ※ | - | - |

※ X APIの無料枠の制約による読み取りエラーです。
